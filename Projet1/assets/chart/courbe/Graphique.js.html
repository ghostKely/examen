Graphique.js

❮ Précédent Suivant ❯

Chart.js est une bibliothèque JavaScript gratuite permettant de créer des graphiques HTML. C'est l'une des bibliothèques de visualisation les plus simples pour JavaScript, et elle est livrée avec les types de graphiques intégrés suivants :

  • Nuage de points
  • Graphique en ligne
  • Diagramme à bandes
  • Diagramme circulaire
  • Graphique en anneau
  • Graphique à bulles
  • Graphique en aires
  • Carte radar
  • Graphique mixte

Comment utiliser Chart.js ?

Chart.js est facile à utiliser.

Tout d'abord , ajoutez un lien vers le CDN (Content Delivery Network) fournisseur :

<script
src="https://cdnjs.cloudflare.com/ajax/libs/Chart.js/2.9.4/Chart.js">
</script>

Ensuite , ajoutez un <canvas> à l'endroit où vous souhaitez dessiner le graphique :

<canvas id="myChart" style="width:100%;max-width:700px"></canvas>

L'élément canvas doit avoir un identifiant unique.

C'est tout!

Syntaxe typique des diagrammes à nuage de points :

var myChart = new Chart("myChart", {
  type: "scatter",
  data: {},
  options: {}
});

Syntaxe typique des graphiques en courbes :

var myChart = new Chart("myChart", {
  type: "line",
  data: {},
  options: {}
});

Syntaxe type d'un graphique à barres :

var myChart = new Chart("myChart", {
  type: "bar",
  data: {},
  options: {}
});

Graphiques linéaires

Code source

var xValues = [];
var yValues = [];
generateData("x * 2 + 7", 0, 10, 0.5);

new Chart("myChart", {
  type: "line",
  data: {
    labels: xValues,
    datasets: [{
      fill: false,
      pointRadius: 1,
      borderColor: "rgba(255,0,0,0.5)",
      data: yValues
    }]
  },
  options: {...}
});

function generateData(value, i1, i2, step = 1) {
  for (let x = i1; x <= i2; x += step) {
    yValues.push(eval(value));
    xValues.push(x);
  }
}
